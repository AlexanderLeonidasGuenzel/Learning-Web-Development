Typography

CSS Typography

Typography is the way that text is arranged and presented.

The form doesn't inherit the font size from the body!

Font Families

  • serif
  • sans-serif
  • monospace
  • cursive
  • fantasy

These are generic families which are usually used as a fallback. As an example:

font-family: 'Courier New', Courier, monospace

First it should take Courier New, but if the device doesn't have this, it looks for Courier, and finally for monospace, if the other fonts are not available.

External Fonts

Go to Website like Google Fonts. Search for a font like "Roboto". Then select the font-weight and font style. You can just use one like Regular 400, or more.

With Link

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

Put these in the <head> of your html file. Important: Insert it above the link to your own stylesheet.

After that go to your css and apply the new font.

.roboto-regular {
   font-family: "Roboto", sans-serif;
   font-weight: 400;
   font-style: normal;
}

If you choose more than one style, your embedded link gets longer and more css rules are to apply.

Notice: The more font styles you attempt to load in from google the longer your page will take to load.

With @import

You can also chooose to use @import and apply the font directly into your css. It is much shorter than via link tag.

<style>
    @import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');
</style>

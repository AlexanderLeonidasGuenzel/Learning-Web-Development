Applying Style

Stylesheets

  • External
  • Internal
  • Inline

The cascading order can change if you put the <link> tag after the <style> tag. Try to avoid inline css. Best way is to use external stylesheet.

External

Style is in a separte .css file and linked via:

<link rel="stylesheet" href="">

Internal

Style is in the .html file in the head.

<head>
  <style>
   p {
       color: limegreen;
   }
  </style>
</head>

Inline

Style as an attribute.

<p style="color: blue"></p>

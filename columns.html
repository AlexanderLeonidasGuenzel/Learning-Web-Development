Column Width 250px

With the

{
  column-count: 4;
  column-width: 250px;
}

property as well. The column width should essentially never be less than 250px per column. Just if it has enough space, it will go up to 4 columns. It could support 1 to 4 colums, that depends on the browsers width.

Shorthand for that is:

{
  columns: 4 250px;
}

CSS Selectors

Overview

  • Element Selector
  • Class
  • ID

Element Selector

body, p, h1, section, ...
{
  property: value;
}

Class Selector

.className
{property: value;}

Classes are more specific and overrule the element selector. They are the most common type of selector with css.

ID Selector

#idName
{property: value;}

Most specific and should only exist once in an html document, beeing unique. Rarely use it for CSS, best for JavaScript to address an element.

Multiple Selection and Inheritance

Grouping

You can address multiple selectors at once, with a comma between.

h1, h2, ...
{property: value;}

Nesting

All <span> in a <p> are selected.

p span
{property: value;}

Universal Selector

It selects everything on the page!

*
{property: value;}

It is typically used for a css reset.

Inheritance

An element can inherit the settings or properties from its parent elment. All other elements inherit the font-size from body.
Anything related to font or typography is inherited! E.g. color, line height, alignment.
Other properties like a border, are not inherited.

body {
   font-size: 22px;
}
 body {
   font-size: 22px;
}

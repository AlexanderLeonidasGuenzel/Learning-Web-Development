Colors

Color Codes

RGB

This represents the color black.

p {
   color: rgb(0,0,0);
}

RGBA

Last value is alpha. This represents the opacity from 0 - 1.

p {
   color: rgb(0,0,0,1);
}

Hex

p {
   color: #19588F;
}
 p {
   color: #FFFFFF;
}

Shorthand is possible if it has pairs.

p {
   color: #FFF;
}

HSL

hue, saturation, lightness

p {
   color: hsl(39, 100%, 75%);
}

Colornames

p {
   color: black;
}

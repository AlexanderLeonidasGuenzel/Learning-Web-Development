Flexbox

Fundamentals

Container

Its a parent elment with property display: flex.

Item

Direct child element of the container. A child could be a container as well.

Properties

Container Properties

  • flex-direction: row or column
  • flex-wrap: nowrap, wrap, wrap-reverse
  • justify-content: flex-start, flex-end, center, space-between, space-around, space-evenly
  • align-items: stretch, flex-start, flex-end, center, baseline
  • align-content: flex-start, flex-end, center, space-between, space-around, space-evenly

Item Properties

  • order: can change the place of an item
  • flex-grow: ability to grow to fill available space
  • flex-shrink: ability to shrink if necessary
  • flex-basis: initial size of the item before space distribution
  • flex: shorthand for flex-grow, flex-shrink, flex-basis together
  • align-self: individual alignment of an item along the cross axis

Usefull

By default it is flex: 0 1 auto

use instead flex: 1 1 0 (grow shrink basis)

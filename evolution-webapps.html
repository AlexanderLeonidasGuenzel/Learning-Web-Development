XMLHTTP - AJAX - Fetch

  • Since 2002
  • Makes autofill possible
  • Outlook web access wanted a component for Network requests without having to reload the page
  • Every time you press a key your browser is sending a request and getting fresh suggestions every
  • 2005 AJAX (Asynchronous JavaScript and XML) as result of XMLHTTP.
  • 2015 API fetch() was implemented. Background requests

Single Page Apps

  • Since 2010
  • REST API

  • React.js Angular JavaScript Frameworks

Websockets

  • Since 2010
  • Usually the browser has first sends a request to the server.
  • With WebSockts the browser connects to the server and it just leaves the connection open.
  • It is lightweight and long lived. The server can tell the browser that something happened.
  • Chat applications.Realtime collaboration. Google Docs, Sheets, Myro, multiplayer gaming.

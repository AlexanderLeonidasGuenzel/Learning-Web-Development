Box Model

What is the CSS box model?

The CSS Box Model is a fundamental concept that defines the structure and layout of HTML elements.

Four main components

  • Content
  • Padding
  • Border
  • Margin

Box Sizing

Content Box

You can see the box model in the browser's inspector in the computed section. The default setting is content-box. This setting applies to the size of the content only. A width of 400px would just be for the content, excluding padding and border. Therefore, it isn't the total width of the element.

Border Box

This setting includes padding and border in the width calculation, but not the margin. It is beneficial to include this setting in a CSS reset or normalization using the universal selector.

{
  box-sizing: border-box;
}

Additional Content

Outline

The outline is not part of the box model and is not calculated in the box model equation.

.outline-offset-positive {
   outline: 5px solid purple;
   outline-offset: 0.5em;
}
.outline-offset-negative {
   outline: 5px solid purple;
   outline-offset: -1em;
}

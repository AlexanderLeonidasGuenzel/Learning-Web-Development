CSS Order

It is important to take care of the order of css rules. Otherwise some properties are overridden.

a {
    text-decoration: none;
    cursor: pointer;
    color: blue;
}
a:visited {
    color: purple;
}
a:hover, a:focus {
    color: dodgerblue;
}

You should use the focus because it is important for screenreaders. Don't forget accessibility!

a:active {
    color: red;
}
